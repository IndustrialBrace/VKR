# Замечания и рекомендации по главе 3 (рабочий журнал)

> Файл создан как служебная разметка для доработки главы 3 ВКР «Разработка стратегии продвижения биометрического продукта Face2 ПАО „АК БАРС“ Банк». В основной документ не возвращать. Стилевой образец — `Замечания_по_главе_2.md`. Анализируемый объект — файл `3 глава.docx` в текущем виде: фактически в нём написан только параграф 3.1 (с дефектами), параграфы 3.2 и 3.3 отсутствуют, хотя заявлены в оглавлении главы и в выводе по главе 2.

## Часть 1. Замечания к структуре главы 3 (по оглавлению в начале документа)

### 1.1. Оценка логики построения

1. Архитектура главы канонически правильная — «обоснование (3.1) → разработка (3.2) → проверка (3.3)». Это совпадает с эталонной структурой практической главы ВКР по маркетингу и с трёхтактной логикой «диагноз → решение → проверка».
2. Структура замкнута на главу 2: 3.1.1 («Проблемы продвижения, выявленные по результатам анализа») явно опирается на сводный перечень из таблицы 2.3.6 (восемь пробелов в трёх кластерах), а 3.3 закрывает контур через KPI и риски.
3. Внутри 3.1 шаги «проблемы → цель → сегменты → позиционирование» логически выстроены как последовательная редукция: от широкого диагноза к формуле позиционирования.
4. В 3.3 объединены четыре уровня проверки (KPI, экономический эффект, риски, оргплан внедрения), что соответствует требованиям к практической части по Положению о ВКР.

### 1.2. Структурные риски

5. Параграф 3.1 переразвит относительно 3.3 и не должен превышать 3.2 по объёму — в практической ВКР по маркетингу основным по объёму должен быть параграф разработки (3.2).
6. В 3.1 нет подраздела с методическим аппаратом стратегии — нет явного моста к моделям главы 1 (STP, расширенный 7P, маркетинговая воронка с этапом «Доверие», push/pull, диффузия Роджерса). Без этого 3.1 рискует быть прочитан как пересказ главы 2.
7. В 3.2 направления (3.2.1) и каналы (3.2.3) разнесены, между ними помещены отраслевые офферы (3.2.2). Это разрывает естественную логику «направления → каналы (как реализуем) → офферы (что говорим)». Сейчас офферы появляются раньше каналов их доставки.
8. 3.2.4 «Контентная и PR-стратегия» по содержанию — частный случай каналов и направлений; в текущем расположении дублирует часть 3.2.3 и 3.2.1.
9. 3.2.5 «Этапы реализации стратегии» и 3.3.4 «Организационный план внедрения» по названиям дублируют друг друга. Этапы и оргплан — это календарь и распределение ответственности, и при текущем расположении они дробятся между двумя параграфами с риском содержательного повтора.
10. В 3.2 нет подраздела о бюджете мероприятий, при том что 3.3.2 («Оценка экономического эффекта») невозможна без CAPEX/OPEX, посчитанных в 3.2.

### 1.3. Взаимосвязь между параграфами

11. Связка 3.1 ↔ глава 2 декларативная: нет вводного фрагмента, в котором перечислены восемь пробелов из таблицы 2.3.6 с указанием, какие из них закрываются в 3.2, а какие выходят за рамки работы.
12. Связка 3.1 ↔ 3.2 не сформулирована: после 3.1.4 (позиционирование) сразу идёт 3.2.1 (направления), но между ними отсутствует мостовой подраздел или матрица «сегмент × ценностное предложение → направление продвижения».
13. Связка 3.2 ↔ 3.3 разорвана через бюджет: нельзя считать ROI стратегии (3.3.2), не сформулировав бюджет мероприятий в 3.2.
14. Связка 3.3.3 ↔ 3.1 не зафиксирована: риски стратегии должны быть сшиты с барьерами биометрии и недоверием (введёнными в 3.1.1), но в текущей структуре эта связь не обозначена.
15. Связка «Вывод по главе 3» ↔ заключение работы не прописана — структура не показывает, где в выводе по главе 3 будет дан перенос результатов в заключение ВКР.

### 1.4. Достаточность раскрытия темы

Тема ВКР содержит три ключевых смысловых узла, и не все они раскрыты структурой:

| Узел темы | Раскрытие в структуре главы 3 |
|---|---|
| «Стратегия» | Раскрыто частично — есть сегменты и позиционирование, но нет подраздела с выбором типа стратегии (push/pull/интегрированная/партнёрская из таблицы 1.2.2) |
| «Продвижение» | Раскрыто плотно в 3.2 (5 подразделов) |
| «Новый / инновационный продукт» | Не раскрыто — ни в одном подразделе нет учёта стадии диффузии Роджерса, хотя для биометрического продукта на стыке коммерциализации и масштабирования это ключевой содержательный вопрос |

16. Не раскрыт инновационный аспект темы. Решение — отдельный подраздел в 3.1 («Стадия диффузии Face2 и приоритетная аудитория стратегии») или явный методический заход в 3.1.3.
17. Не раскрыт выбор типа стратегии. Подраздел 3.1.4 называется «Позиционирование», но позиционирование — это элемент стратегии, а не сама стратегия. Нужен подраздел с выбором между альтернативами и обоснованием одной из них.
18. Не раскрыта двухуровневая модель продаж Face2 (КБС под ключ оператору / Face2Pay-Pass-Check-in конечному объекту), введённая в таблице 2.1.4 главы 2. Без её отражения в 3.1 или 3.2 существенная часть аналитической находки главы 2 «теряется» при переходе к разработке.

## Часть 2. Замечания к параграфу 3.1 (фактически написанному тексту)

### 2.1. Структурные дефекты параграфа

27. **Тройная заголовочная конструкция в начале.** В тексте подряд идут три разных названия главы 3 («3. РАЗРАБОТКА СТРАТЕГИИ ПРОДВИЖЕНИЯ НОВОГО ПРОДУКТА АК БАРС БАНКА», «3.1. Формирование целей и направлений продвижения», «Глава 3. Разработка стратегии продвижения инновационного продукта Face2») и внутреннее оглавление главы. Это рабочая, не сведённая чистовая версия. По Положению (п. 10.1) глава начинается с одного заголовка, параграф — с одного заголовка.
28. **Нет внутренних подзаголовков подразделов 3.1.1–3.1.4.** Подразделы заявлены в оглавлении главы, но в самом тексте параграфа не размечены — читателю приходится восстанавливать границы по контексту.
29. **Авторская пометка «НИЖЕ ДОБАВИТЬ ПРО B2C» в чистовом тексте.** Маркер несведённой версии. Должна быть удалена.
30. **Ошибочная нумерация одной из таблиц.** Таблица «Цели и задачи стратегии продвижения Face2» помечена как «Таблица 3.2.2», при том что находится в параграфе 3.1. Корректный номер — 3.1.2.
31. **Незаполненные источники под таблицами.** Под всеми четырьмя таблицами (3.1.1, «3.2.2», 3.1.3, 3.1.4) стоит «Источник: составлено автором на основе» с пустым продолжением. Положение (п. 10.2) требует указания источника полностью.
32. **Отсутствие визуализации.** В стратегическом параграфе напрашивается одна сводная схема (цель → задачи → направления → переход к мероприятиям 3.2). Сейчас её нет.

### 2.2. Логические дефекты

33. **Цель параграфа сформулирована дважды и в разных скоупах.** В первом упоминании (вводный блок) цель — «B2B и B2G-клиенты»; в развёрнутой формулировке после таблицы 3.1.1 — «B2B, B2С и B2G-клиенты». При этом сохранена пометка «НИЖЕ ДОБАВИТЬ ПРО B2C». Скоуп стратегии «плавает» внутри одного параграфа.
34. **Диагноз проблем в 3.1.1 не сшит с таблицей 2.3.6 главы 2.** В тексте перечислены 5 проблем, в таблице 3.1.1 — 7 строк, в таблице 2.3.6 главы 2 — 8 пробелов. Какие из 8 пробелов главы 2 закрывает каждая из 5/7 проблем 3.1.1 — нигде не показано.
35. **Тройное повторение сегментных ценностей.** Содержание таблицы 3.1.3 (столбец «Ключевой акцент в продвижении»), буллет-список «Ключевые ценности Face2 для сегментов» и абзац «для банка — безопасность, для ритейла — скорость…» дают одну и ту же информацию в трёх формах подряд.
36. **Двойная формула позиционирования.** Формула «Face2 — это комплексная отечественная биометрическая платформа…» вводится в подразделе 3.1.4 дважды разными словами (сначала во вводном абзаце, потом отдельным выделенным определением). По правилам учебной работы формула вводится один раз.
37. **Слитные несимметричные перечни «продукты» и «отрасли».** Перечень из 9 продуктов линейки и перечень из 11 отраслей применения идут друг за другом без явного разделителя. Это разные сущности (что продаём / кому продаём), и их соединение должно быть матричным, а не последовательным.
38. **Смешение разноуровневых вещей в перечне проблем 3.1.1.** В одном списке смешаны: барьеры биометрии (внешние, рыночные — недоверие, страхи утечек), особенности продукта (внутренние — сложность линейки) и собственно дефициты продвижения (узнаваемость, доказательная база). Корректнее развести три класса и работать в стратегии прежде всего с третьим.
39. **Цель не SMART.** Нет горизонта (на сколько лет), нет численного критерия достижения (рост узнаваемости — на сколько). По требованиям к практической части ВКР цель должна быть измеряемой.
40. **Сегмент B2C, заявленный в развёрнутой цели, как сегмент не разработан** — для него ни одной задачи отдельно нет, отдельного подраздела или абзаца нет.

### 2.3. Содержательные дефекты — связь с теоретической базой главы 1

41. **Не использована модель STP.** В подразделах 3.1.3 и 3.1.4 фактически реализованы шаги T (выбор сегментов) и P (позиционирование), но без явной отсылки к модели и без шага S (сегментации как методического действия). Сама модель не названа.
42. **Не использован комплекс 7P.** Параграф 3.1.4 (позиционирование) должен соотноситься с целевой конфигурацией 7P, а в 2.3 уже сделан 7P-разбор Face2 «как есть» (таблица 2.3.3). Логично было бы сделать сравнительную таблицу «как есть → как должно быть» — её нет.
43. **Не использована маркетинговая воронка с этапом «Доверие».** В главе 1 (таблица 1.2.4) специально подчёркнуто, что для биометрического продукта обязателен этап «Доверие» между интересом и пробным использованием. В главе 2 (рис. 2.3.1, таблица 2.3.4) построена «фактическая» воронка Face2. Глава 3 должна предложить, как этот этап будет наполняться, но в 3.1 воронка не упоминается.
44. **Не введена теория диффузии инноваций Роджерса.** Категории «новаторы / ранние последователи / раннее большинство / позднее большинство / отстающие» в параграфе 3.1 не названы, хотя для Face2 (особенно для B2C-сценариев — метро, ритейл, гостиница) разделение аудитории по стадиям принятия инновации — содержательно ключевое.
45. **Не использована рамка push/pull-стратегий.** В выводе главы 2 (через SWOT и сводный вывод) явно зафиксирован диагноз «push сильный, pull фрагментарный».
