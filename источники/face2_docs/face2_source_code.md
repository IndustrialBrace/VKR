## Требования при разработке исходного кода

В рамках соблюдения правил и требований по разработке отечественных программных продуктов сформирован принцип, по которому в любой момент времени исходный код разрабатываемого ПО находится на территории Российской Федерации.

На основании этого были внесены изменения в должностные инструкции:

- о необходимости вести разработку программных продуктов на территории РФ;

- в том числе хранить исходный код ПО исключительно на серверах Компании;

- на момент отпуска сотрудника с выездом за границу РФ оборудование Компании должно быть оставлено в офисе Компании.

Для хранения исходного кода используется программный продукт GitLab, представляющий собой систему контроля версий для хостинга проектов и их совместной разработки. GitLab развернут на выделенном виртуальном сервере в ЦОД, доступ к GitLab возможен только с использованием VPN компании.

## Описание исходного кода

Исходные текст «Face2» написан с использованием следующих языков программирования: C#, C++, JavaScript, Python. Для верстки интерфейса используется язык разметки HTML5.

Компиляция исходного кода ПО «Face2» производится на серверах, расположенных в Российской Федерации.

В качестве технического средства компиляции исходного кода ПО «Face2» в объектный код используется утилита dotnet build и компилятор Roslyn для C#.

Интерфейс ПО собирается с помощью утилиты webpack, так как язык JavaScript не предполагает предварительной компиляции в объектный код.

Микросервисы Системы доставляются в виде Docker-образов средствами CI/CD или вручную.

## Лицензионная модель

Система «Face2» разрабатывается с поддержкой лицензионной модели эксплуатации. В качестве лицензионных ключей используется программный код, разработанный внутри Компании под индивидуальные потребности продукта и может представлять из себя физический носитель с защищенным от копирования пространством или зашифрованный файл, содержащий в себе необходимую информацию.

Система «Face2» распространяется по лицензионной модели с ограничением по времени и количеству устройств биометрических терминалов, которые можно подключить в период использования. Выпуск, распространение, управление лицензионными ключами программного обеспечения «Face2» осуществляется в рамках договора на сопровождение «Face2» командой технической поддержки.
